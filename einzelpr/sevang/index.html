Was ist ein
Template String?


Template Strings benutzen backticks (``)

1  //Normale Strings:
2  var str1 = "blabla";
3  var str2 = 'blabla';
4  
5  //Template String:
6  var str3 = `blabla`;

Was können
Template Strings?


  • Mehrzeilige Strings
  • Ausdrücke einbetten
  • Tagged Template Strings

Mehrzeilige Strings

Code

1  var str = `String Zeile 1
2  String Zeile 2`;


Konsole

String Zeile 1
String Zeile 2

Ausdrücke einbetten

Code

1  var name = "Sevan";
2  
3  console.log(`Mein Name ist ${name}.`);


Konsole

Mein Name ist Sevan.

Tagged Template Strings

Code

1  var name = "Sevan";
2  var alter = 19;
3                      
4  function tag(strings, ...ausdrücke) {
5      console.log(strings);
6      console.log(ausdrücke);
7  }
8  
9  tag `Mein Name ist ${name} und ich bin ${alter} Jahre alt.`;

Konsole

▼Array[3]
    0: "Mein Name ist "
    1: " und ich bin "
    2: " Jahre alt."
    length: 3
▼Array[2]
    0: "Sevan"
    1: 19
    length: 2

Im Vergleich

Gewünschte Konsolenausgabe

Eine ganz einfache Addition:

Das Ergebnis von a(=12) addiert mit b(=10) ist 22.

Normaler String

1  var a = 12;
2  var b = 10;
3        
4  console.log("Eine einfache Addition:\n\nDas Ergebnis von a(=" + a + ") addiert mit b(=" + b + ") ist " + (a + b) + ". ");

Template String

1  var a = 12;
2  var b = 10;
3           
4  console.log(`Eine einfache Addition:
5
6  Das Ergebnis von a(=${a}) addiert mit b(=${b}) ist ${a+b}.`);
